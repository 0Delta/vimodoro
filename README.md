# Vimodoro

Vimodoro is pomodoro timer for Vim.

## Usage

 * `:Pomodoro`  
   Start pomodoro timer.
 * `:PomodoroStart`  
   Stop pomodoro timer. 

## Installation

```
# with dein
[[plugins]]
repo = '0Delta/vimodoro'

# with NeoBundle
NeoBundle "0Delta/vimodoro"
```
